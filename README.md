# Elefanto tech assignment

Rest API for book store

## Installation

Make sure you have [Docker](https://www.docker.com/products/docker-desktop/) to run the app.

```bash
make build
make up
```

## Additinal make features

```python
make make_migrations
make lint
make test
make create_superuser
make add_app

```
